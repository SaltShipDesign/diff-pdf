# Go into the host using terminal / ssh
# Go into the home directory (/home/<user>)
>> snap install docker
>> docker pull ubuntu
# Copy the docker-root folder from the repo to the user home folder.
# Go into the destination folder, e.g. /home/<user>/docker-root
>> docker build -t diff-pdf-image .
# Copy test pdf files into /home/<user>/testpdfs
# Go into the folder /home/<user>/testpdfs
>> docker run --rm -v <mount folder full path>:/data diff-pdf-image <pdf-file-1> <pdf-file-2> <output-pdf>

Example:
>> docker run --rm -v D:\Temp\_diff-pdf\pdfs:/data diff-pdf-image pdf_A.pdf pdf_B.pdf custom-diff.pdf
>> docker run --rm -v /home/salt/testpdfs:/data diff-pdf-image 0217-101-001-C.pdf 0217-101-001-I.pdf diff.pdf

After completion, the <output-pdf> will be available in the mount folder
